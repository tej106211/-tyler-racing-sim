TYLER'S RACING SIM FUND PWA

WHAT THIS IS
- A private installable Progressive Web App bundle.
- It includes the full interactive tracker, local saving, offline caching, and Home Screen app icons.
- Nothing has been published or uploaded by ChatGPT.

IMPORTANT IPHONE LIMITATION
Apple only allows normal PWA installation from Safari when the files are served from an HTTPS web address.
Opening index.html directly from the Files app will show the app, but it will not become a true installable PWA.

FILES
- index.html
- manifest.webmanifest
- sw.js
- icons/

AFTER IT IS HOSTED PRIVATELY OVER HTTPS
1. Open the private HTTPS address in Safari.
2. Tap Share.
3. Tap Add to Home Screen.
4. Tap Add.
5. Open Racing Sim Fund from the Home Screen.

PRIVACY
- The app stores tracker data in that browser/app's local storage.
- The PWA itself does not send your savings data anywhere.
- Use Export Backup inside the app before switching devices or clearing Safari data.

HOSTING NOTE
A private HTTPS host is required for installation. The bundle is ready for any static HTTPS host.
